# 衍生品偏斜情绪盯盘 Agent 交接卡

## 当前回答的问题

期权市场是否正在为风险或事件预期支付明显溢价？

## 当前状态

- 状态标签：`iv-premium-watch`
- 风险等级：`高关注`
- 主要观察焦点：隐含波动率、历史波动率、IV-HV 差和高隐波合约。

## 现在应该怎么用

把 IV-HV 差、高隐波合约和标的历史波动放到同一张观察表。

## 升级与降级

- 升级观察：IV-HV 差继续扩大，且高隐波集中在有解释力的到期或行权结构。
- 降级观察：隐波快速回落，或历史波动率上升解释了大部分溢价。

## 边界

本 Agent 只提供研究、盯盘和复盘输入，不负责交易执行。
